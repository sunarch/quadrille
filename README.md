# quadrille

Square grid simulations

## Installation

Available on PyPI: [quadrille](https://pypi.org/project/quadrille/)

```commandline
pip install quadrille
```

## Parts

- Conway's Game of Life
- Langton Ants
    - Wikipedia: [Langton's ant](https://en.wikipedia.org/wiki/Langton%27s_ant)
